# Documentation

This directory contains the extended documentation for `helm-fine-upgrade`.

## Guides

- [Use Cases](./use-cases.md): choose a workflow for upgrade planning, runtime
  drift checks, resource adoption, and selector migration.
- [Installation And Troubleshooting](./installation-troubleshooting.md): Helm 3
  and Helm 4 install guidance, binary-package notes, and common failure cases.
- [Binary Release](./binary-release.md): how packaged plugin assets work and how
  to install them directly from GitHub Releases.
- [Release Checklist](./release-checklist.md): steps to verify before publishing
  a release tag.
